# Resources

The 2026-03 revision of the Model Context Protocol adds a `_meta` field that is
now accepted on every request and response envelope, replacing the older
per-method extension blocks.

```
{ "jsonrpc": "2.0", "method": "tools/list", "params": {}, "_meta": { "trace": "abc" } }
```

=== SOURCE ===
url: https://github.com/modelcontextprotocol/spec/releases
title: MCP spec releases
date: 2026-04-12

Release 2026-04 clarifies that servers MUST echo the `_meta.trace` value back on
every response so clients can correlate streamed results. Tool result schemas are
otherwise unchanged from the 2026-03 revision.
